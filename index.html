Bootcamp 2020: Milestone3

This game is a 2d game i have created for my milestone 3 project. The purpose of this game is to steal all the cows which are at the top of each platform in all 5 levels. Interacting with the cows will enable the player to move onto the next level. You will have to be careful of the enmies in the game which can include still enemies or enemies which can attack you when you go closer to them. This would cause the player to loose a life and if you loose all your lives the game will be over! Once you have got the last cow the game will end and the player will win.
